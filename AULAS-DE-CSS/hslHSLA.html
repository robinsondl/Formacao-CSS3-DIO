Cores

Existem várias formas de definirmos cores utilizando o CSS. Elas são:

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hs1(hue, saturation, lightness)
  • Hue m(matriz): é o grau na roda de cores (de 0 a 360)
    • 0 ou 360 é vermlho
    • 120 é verde
    • 240 é azul
  • saturation (saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Lightness (luminosidade): luminosidade da cor
    • 0% é preto
    • 100% é branco
